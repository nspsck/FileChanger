# A simple filename changer

This changes all new picutures added to a specific directory/folder with the file extension of one of the following:
- jpg
- jpeg
- png
to a specific `filename.jpg` in the same directory.

## Requirements:

`watchdog`. To install:

```pip3 install watchdog```

On newer linux:
Debian based:
```sudo apt install python3-watchdog```
Arch based:
```sudo pacman -S python-watchdog```

## Usage
Shell:
```python3 ChangeFileName.py```
